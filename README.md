# PureFlow Elite - Premium Air Duct Cleaning Service

A complete, production-ready premium air duct cleaning service website built with Next.js 14, MongoDB, and Tailwind CSS. Features online booking, email notifications, customer dashboard, and a stunning premium UI.

## 🚀 Live Demo
[View Demo](https://premium.pureflowair.demo)

## 📋 Features

### Premium Features
- ✅ **Online Booking System** - 3-step booking with date/time selection
- ✅ **Email Notifications** - Automated confirmation emails via Resend
- ✅ **Customer Portal** - Booking history and account management
- ✅ **Discount Code System** - Support for promo codes (ELITE20)
- ✅ **Admin Dashboard** - Analytics, booking management, and insights
- ✅ **Contact Form** - Lead capture with MongoDB storage
- ✅ **Blog System** - Articles with category filtering
- ✅ **Gallery** - Premium image showcase with lightbox
- ✅ **Services Page** - Detailed service listings with comparison table
- ✅ **FAQ Section** - Accordion-style frequently asked questions

### Technical Stack
- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS with custom premium theme
- **Database:** MongoDB with Mongoose
- **Email:** Resend API for transactional emails
- **Animations:** Framer Motion
- **Icons:** React Icons
- **Language:** JavaScript (ES6+)
